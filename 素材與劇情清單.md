# 臺中驛站 — 素材與劇情統整清單

線上版（GitHub）所有人都看得到的東西，只有兩種：
1. **放在 `assets/` 資料夾、且推上 GitHub 的檔案**
2. **後台匯出、存成 `assets/story.json` 的劇情**

後台「上傳」進瀏覽器的東西**不會**上 GitHub、別人看不到。所以照這份做。

==============================================================
## 一、劇情（不用重寫！用你已經改好的後台）
==============================================================

1. 開你**已經改好劇情的後台** `admin.html`。
2. 上方分頁 **「匯入匯出」** → 按 **「匯出 JSON」**。
3. 會下載一個 `tcgame-2026-xx-xx.json`。
4. 把它**改名成 `story.json`**，放到：
   `C:\Users\user\Desktop\臺中驛站\assets\story.json`
5. 跟我說「story.json 放好了」，我幫你提交上 GitHub。

之後要改劇情：後台改 → 再匯出一次 → 覆蓋 `assets/story.json` → 重新提交。完全不用動 HTML。

==============================================================

**檔名必須完全一致（含副檔名、大小寫）**。打勾代表你已放好。

把對應圖檔放成 `assets/images/<那個key>.png`（或 .jpg）即可，例如：
- [ ] `mem-slider1.png` … `mem-slider8.png`（八段回憶圖，若你的 key 是這樣命名）

- [ ] `photo-final-ref.jpg`（最終拍照比對圖）
